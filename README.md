# Virtual File System

You can vist live demo here: <https://virtual-file-system.vercel.app>

## Installation

```bash
npm install
```

## Running the app

### 1. Create environment file

Create *.env* file at root folder from *.env.template* file

### 2. Run app

```bash
# development
npm run dev
```
